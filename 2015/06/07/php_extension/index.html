PHP 扩展开发笔记

By 严宇川

2015-06-07 更新日期:2015-06-07

以下基于 VirtualBox + Ubuntu 实现
虚拟机中已安装 PHP + Apache
编辑器推荐使用 Sublime Text 3
使用 root 账号, 不然记得加上 sudo

代码目录

cd /
mkdir ever
cd ever

安装编译工具

sudo apt-get install gcc make autoconf

下载PHP源码, 需要用到 ./ext/ext_skel

http://php.net/

下载 php-5.6.9.tar.bz2

解压到目录 ./ever/php-src/下

tar -jxvf xx.tar.bz2

创建默认扩展

进入 ext 目录

cd php-src/ext

ext_skel —extname=test

创建 test 扩展, 将会在当前目录, 也就是 ./ext/ 下新建 test 目录及 C 文件,

cd test

启用扩展

vim config.m4

将 : 

dnl PHP_ARG_ENABLE(test, whether to enable test support,  
dnl Make sure that the comment is aligned:  
dnl [  --enable-test           Enable test support])

更改为 : 

PHP_ARG_ENABLE(test, whether to enable test support,  
[  --enable-test           Enable test support])

编译, 依次执行

phpize
./configure
make
make install

编辑 php.ini, 开启 test 扩展

php -i | grep 'php.ini'
vim /etc/php5/cli/php.ini
extension=test.so

测试

<?php
echo confirm_test_compiled("hello world");

终端执行 : 

php demo.php

开发扩展

编辑 test.c

const zend_function_entry test_functions[] = {
    PHP_FE(confirm_test_compiled,    NULL)        /* For testing, remove later. */
    //新增
    PHP_FE(hello,    NULL)    
    PHP_FE_END    /* Must be the last line in test_functions[] */
};

新增 : 

PHP_FUNCTION(hello)
{
    long a;
    long b;
    char *c;
    int c_len;

    if (zend_parse_parameters(ZEND_NUM_ARGS() TSRMLS_CC, "lls", &a, &b, &c, &c_len) == FAILURE) {
        return;
    }
    char *str;
    int len = spprintf(&str, 0, "%s : %d\n", c,  a*b );
    RETURN_STRINGL(str, len, 0);    

}

重编译

make 
make install

测试

<?php
echo hello(2, 3, "hey");

终端执行 : 

php demo.php

Web 访问 (Apache)

找到 Apache 的 php.ini 文件

vim /etc/php5/Apache/php.ini   
新增 extension=test.so

service apache2 restart

浏览器访问相应文件即可, 可与 phpinfo() 里查看扩展是否有开启
