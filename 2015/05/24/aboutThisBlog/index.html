关于这个 BLOG

我要说一个故事, 这个故事是这样子的.

身为一个悲催的程序猿, 常在河边走, 总也会遇到一些不常见的坑坑洼洼让你摔个大跤.
所以不免从百度到谷歌, 千辛万苦找到了救命良方, 遂存之书签.
但日积月累, 书签越来越多, 然后出现了这样一个过程 :

  • 踩过的坑又踩
  • 搜过的东本又搜
  • 存之书签
  • 踩过坑又踩
  • ……

于是想着弄个博客吧.

然后…

为什么放着那么多国内博客不用, 要用 Github 呢?

  • 有逼格
  • 有逼格
  • 有逼格

听说重要的事要说三遍!!!

好吧, 事情是这样子的.

之前用 Github 仅限于搜一些好的开源代码, 然后你懂的.

后来发现用 Github 写博客于我而言有以下好处 :

  1. 免费
  2. 接触新东西, 像之前有逼格的 Markdown 根本很少用, 然后一用发现像吃了炫迈一样完全停不下来啊
  3. 经常用更有助于加深对 Github 的熟悉度, 之前用 SVN 的说

发现有强迫症, 不满三点要凑足三点.

  • 我不是处女座
  • 我不是处女座
  • 我不是处女座

写在后面, 记住一定要经常更新, 因为是写给自己看的.

恩, 就是这样, 待会就是西甲最后一轮巴萨 VS 拉科鲁尼亚了, 感谢哈维, 加油巴萨.

三冠王我们来了!!!
