Gitbook 笔记

By 严宇川

2016-06-04 更新日期:2016-06-04

Gitbook 个人学习笔记.

1). 安装 Gitbook

  1. 安装 Node.js
  2. 使用 npm 安装 Gitbook
    1. npm install -g gitbook
    2. npm install -g gitbook-cli
    3. type ‘gitbook -V’ 检查 Gitbook 是否安装成功

2). Gitbook 描述文件

  1. README.md

    首页

  2. SUMMARY.md

    目录结构

    • page1
      • chapter1
      • chapter2
    • page2

3). 命令

mkdir demo
cd demo
touch README.md
touch SUMMARY.md

3.1). 初始化

gitbook init
tree [# check the menu and file]

3.2). 本地预览

gitbook serve [http://localhost:4000]

3.3). 生成 HTML

gitbook build demo /tmp/gitbook

3.4). 生成 PDF

npm install -g gitbook-pdf
…

3.5). 帮助

gitbook help

4). Gitbook 配置信息

touch book.json

标识 说明
title 标题
author 作者信息
description 书本描述
language 使用的语言
links 在侧边栏添加链接
styles 自定义样式
plugins 插件
pluginsConfig 插件配置
gitbook 指定gitbook版本

language

en, ar, bn, cs, de, en, es, fa, fi, fr, he, it, ja, ko, no, pl, pt, ro, ru, sv, uk, vi, zh-hans, zh-tw

links

"links" : {
    "sidebar" : {
        "Home" : "http://zhangjikai.com"
    }
}

styles

"styles": {
    "website": "styles/website.css",
    "ebook": "styles/ebook.css",
    "pdf": "styles/pdf.css",
    "mobi": "styles/mobi.css",
    "epub": "styles/epub.css"
}

5). Gitbook 插件

https://plugins.gitbook.com
gitbook install 安装插件

默认插件

  • highlight
  • search
  • sharing
  • font-settings
  • livereload

移除默认插件

plugins: [
    "-sharing"
]

插件配置

"pluginsConfig": {
    "fontsettings": {
        "theme": "sepia",
        "family": "serif",
        "size":  1
    }
}

常用插件

Duoshuo 评论系统

https://plugins.gitbook.com/plugin/duoshuo

中文搜索

https://plugins.gitbook.com/plugin/search-pro

目录折叠

https://plugins.gitbook.com/plugin/expandable-chapters

图片查看

https://plugins.gitbook.com/plugin/fancybox

左侧TAB栏动态切换宽度

https://plugins.gitbook.com/plugin/splitter

统一尾部

https://plugins.gitbook.com/plugin/tbfed-pagefooter

捐赠模块

https://plugins.gitbook.com/plugin/donate

Github 链接

https://plugins.gitbook.com/plugin/github

emoji 表情

https://plugins.gitbook.com/plugin/advanced-emoji

katex 数学公式

https://plugins.gitbook.com/plugin/katex

Mermaid 图表

https://plugins.gitbook.com/plugin/mermaid
