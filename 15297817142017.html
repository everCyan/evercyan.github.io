PHP 扩展开发备注

2016/5/1 posted in  Note

基于 MAC OS.
本次扩展名称为 demo, 最终结果会生 demo.so, 并被 php.ini 加载.

准备工作

  • 安装编译工具
    sudo apt-get install gcc make autoconf
  • PHP官网下载相应版本 http://php.net/downloads.php
  • 解压下载好的 PHP 代码, 本次测试在 ~/Documents/php-7.0.10/

生成扩展

  • cd ~/Documents/php-7.0.10/ext/
  • ./ext_skel --extname=demo
  • cd demo

  • vim config.m4

    将
    dnl PHP_ARG_ENABLE(test, whether to enable test support,
    dnl Make sure that the comment is aligned:
    dnl [ --enable-test Enable test support])
    三行前面的 dnl 干掉

  • phpize

  • ./configure

  • sudo make

  • sudo make install

  • ls -alt ./modules
    目录下会有一个 demo.so
    全路径为 /User/Cyan/Documents/php-7.0.10/ext/demo/modules/demo.so

加载扩展并执行

  • php -i | grep 'php.ini'
    查找当前生效的 PHP 配置文件, 本机为 Applications/MAMP/bin/php/php7.0.10/conf/php.ini
  • vim Applications/MAMP/bin/php/php7.0.10/conf/php.ini
    添加扩展
    extension=/User/Cyan/Documents/php-7.0.10/ext/demo/modules/demo.so

验证

php -m | grep 'demo'
php -r 'echo phpinfo();' | grep 'demo'
php -r 'echo confirm_demo_compiled("demo");'
终端输出 : Congratulations! You have successfully modified ext/lodash/config.m4. Module lodash is now compiled into PHP.

最后

以上仅为默认开发扩展的步骤, 具体扩展功能的实现请参考学习 http://www.cunmou.com/phpbook/index.md
